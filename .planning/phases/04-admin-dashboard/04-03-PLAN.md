- If `editingFaq` is non-null (editing): We need a PUT/PATCH for editing. Since the API route defined in Plan 01 has DELETE but not a dedicated UPDATE — implement edit as DELETE old + POST new, OR add a PATCH handler in the API. **Simpler approach:** Delete old FAQ, create new one. `await fetch(\`/api/config/${botId}/faqs\`, { method: 'DELETE', body: JSON.stringify({ faqId: editingFaq.id }) })` then POST new one. On success: `toast.success('FAQ updated.')`, close dialog, refetch.
- On error: `toast.error('Failed to save. Please try again.')`
- `setSaving(false)`

**IMPORTANT UPDATE:** The executor should add a PATCH export to `app/api/config/[botId]/faqs/route.ts` that accepts `{ faqId, question, answer, language }` and does `.from('faqs').update({ question, answer, language }).eq('id', faqId).eq('bot_id', botId)`. Then use PATCH for editing instead of delete+create. This keeps IDs stable.

**handleDelete:**
- `fetch(\`/api/config/${botId}/faqs\`, { method: 'DELETE', body: JSON.stringify({ faqId }) })`

```typescript
const grouped = INTENTS.map(intent => {
```

**Open edit dialog:** Given an intent_key, find its EN/BM/ZH content from grouped data:
```typescript
function openEditDialog(intent: string) {
  const variants = templates.filter(t => t.intent_key === intent)
  setEditingIntent(intent)
  setFormEn(variants.find(v => v.language === 'en')?.content ?? '')
  setFormBm(variants.find(v => v.language === 'bm')?.content ?? '')
  setFormZh(variants.find(v => v.language === 'zh')?.content ?? '')

}
```

**handleSaveTemplate:**
- `setSaving(true)`
- Save all three variants sequentially (or Promise.all):
  ```typescript
  const saves = LANGUAGES.map(lang => {
    const content = { en: formEn, bm: formBm, zh: formZh }[lang]
    if (!content.trim()) return Promise.resolve() // skip empty variants
    return fetch(`/api/config/${botId}/templates`, {
      method: 'PATCH',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ intent_key: editingIntent, language: lang, content }),

  })
  ```
