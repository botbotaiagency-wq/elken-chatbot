**Create app/api/ingest/[botId]/route.ts — POST handler for upload initiation:**

1. Import `createServiceClient` from `@/lib/supabase/service`
2. Accept JSON body: `{ filename: string, category: DocumentCategory, contentType: string }`
3. Validate contentType is one of: `application/pdf`, `application/vnd.openxmlformats-officedocument.wordprocessingml.document`, `text/plain`
4. Validate category is one of: `Beauty`, `FMCG`, `GenQi`, `Healthfood`, `Home Appliances`, `Other`
5. Insert into `documents` table: `{ bot_id: params.botId, filename, category, status: 'pending' }`
6. Generate storage path: `${params.botId}/${doc.id}/${filename}`
7. Call `supabase.storage.from('bot-files').createSignedUploadUrl(storagePath)`
8. Update document with `storage_path`
9. Return `{ documentId: doc.id, signedUrl, token, path: storagePath }`
10. Add `export const maxDuration = 60` at top of file
11. On any error, return `Response.json({ error: message }, { status: 400 or 500 })`

**Create app/api/ingest/[botId]/process/route.ts — POST handler for processing pipeline:**

1. Import extractText, chunkText, embedDocumentChunks from `@/lib/ingest/*`
2. Import `createServiceClient` from `@/lib/supabase/service`
3. Accept JSON body: `{ documentId: string }`
4. Fetch document record from `documents` table by id — verify `bot_id === params.botId`
5. Set document status to `processing`
6. Download file from storage: `supabase.storage.from('bot-files').download(doc.storage_path)`
7. Extract text: `const text = await extractText(Buffer.from(await blob.arrayBuffer()), contentType)`
   - Determine contentType from filename extension: `.pdf` -> `application/pdf`, `.docx` -> `application/vnd.openxmlformats-officedocument.wordprocessingml.document`, `.txt` -> `text/plain`
8. Chunk: `const chunks = chunkText(text, 500, 50)`
9. If chunks.length === 0, set status='failed', error_message='No text content extracted', return error
10. Embed: `const embeddings = await embedDocumentChunks(chunks.map(c => c))`
    - Batch in groups of 128 (handled by embedder internally)
11. Bulk insert into `chunks` table: for each chunk[i], insert `{ bot_id: params.botId, document_id: documentId, content: chunks[i], embedding: embeddings[i] }`
12. Update document: `{ status: 'ready', chunk_count: chunks.length }`
13. On any error during processing: set document `{ status: 'failed', error_message: error.message }`, return 500
14. Add `export const maxDuration = 60` at top of file
15. Wrap entire processing in try/catch — failures MUST update document status to 'failed'

**Create app/api/documents/[botId]/route.ts — GET handler for listing:**

1. Query `documents` where `bot_id = params.botId`, order by `created_at DESC`
2. Return array of document objects (id, filename, category, status, chunk_count, error_message, created_at)

**Create app/api/documents/[botId]/[documentId]/route.ts — DELETE handler:**

1. Fetch document by id, verify `bot_id === params.botId`
2. Delete from storage: `supabase.storage.from('bot-files').remove([doc.storage_path])` (if storage_path exists)
3. Delete from `documents` table — cascade deletes all chunks automatically (FK constraint)
4. Return `{ success: true }`
